# AI翻译输入法 - 打包说明

## 快速打包

### 方法一：使用批处理脚本（推荐）

1. 双击运行 `build_installer.bat`
2. 等待打包完成
3. 在 `dist\AI翻译输入法\` 目录下找到可执行文件

### 方法二：手动打包

```powershell
# 1. 安装 PyInstaller
python -m pip install pyinstaller pyinstaller-hooks-contrib -i https://mirrors.aliyun.com/pypi/simple/

# 2. 安装项目依赖
python -m pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/

# 3. 执行打包
python -m PyInstaller build.spec --noconfirm --clean
```

## 创建安装程序

打包完成后，需要使用 Inno Setup 创建安装程序：

1. **下载安装 Inno Setup**
   - 官网：https://jrsoftware.org/isinfo.php
   - 下载 Unicode 版本

2. **编译安装程序**
   - 用 Inno Setup 打开 `installer.iss`
   - 点击菜单 `Build` → `Compile`
   - 等待编译完成

3. **获取安装包**
   - 安装程序在 `installer_output` 文件夹
   - 文件名：`AI翻译输入法_安装程序_v1.0.0.exe`

## 常见问题

### Q: 打包后运行提示缺少 DLL？

A: 确保已安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### Q: 打包后程序无法启动？

A: 尝试以下步骤：
1. 在命令行运行 exe 查看错误信息
2. 检查是否缺少 `config.json`（首次运行会自动创建）
3. 确保有管理员权限（热键功能需要）

### Q: 如何添加程序图标？

1. 准备一个 `.ico` 格式的图标文件
2. 重命名为 `app.ico` 放在项目根目录
3. 重新运行打包脚本
