使用方法

三步上手，立即开始高效沟通

1

配置 API 密钥

首次使用需要配置 AI 服务的 API 密钥。点击设置图标，选择您使用的 AI 服务商（如 DeepSeek、OpenAI 等），填入 API Key 即可。

💡 推荐使用 DeepSeek，性价比高
2

锁定聊天窗口

将光标放在目标聊天窗口的输入框中，然后按 Alt + T 呼出翻译窗口，点击「锁定」按钮锁定当前聊天窗口。

💡 锁定后窗口名称会显示在顶部
3

输入翻译发送

在输入框中输入中文内容，按下 Enter 键，翻译结果会自动粘贴到锁定的聊天窗口并发送。

💡 可在设置中选择是否自动发送

常见问题

快速解决您的疑问

如何获取 API Key？ ▼
推荐使用 DeepSeek：访问 platform.deepseek.com 注册账号，在控制台创建 API Key 即可。DeepSeek 提供免费额度，且价格实惠。也可使用 OpenAI、通义千问等其他支持 OpenAI 格式的 API。
支持哪些聊天软件？ ▼
支持所有 Windows 桌面聊天软件，包括但不限于：钉钉、微信、企业微信、WhatsApp、Telegram、Skype、阿里旺旺、千牛等。只要是可以粘贴文字的输入框都可以使用。
翻译质量如何？ ▼
翻译质量取决于您使用的 AI 模型。推荐使用 GPT-4、DeepSeek 等大语言模型，翻译质量远超传统机器翻译，能准确理解上下文和电商专业术语。
软件是否安全？ ▼
软件完全本地运行，不收集任何用户数据。翻译请求直接发送到您配置的 AI 服务商（如 DeepSeek、OpenAI），不经过任何第三方服务器。API Key 仅保存在您的电脑本地。
可以修改快捷键吗？ ▼
可以。点击设置图标，在「快捷键」选项卡中可以自定义呼出窗口的快捷键。支持 Alt、Ctrl、Shift 等组合键。
