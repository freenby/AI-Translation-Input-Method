How to Use

Get started in three simple steps

1

Configure API Key

First-time setup requires an AI service API key. Click the settings icon, select your AI provider (DeepSeek, OpenAI, etc.), and enter your API Key.

💡 Recommend DeepSeek for best value
2

Lock Chat Window

Place your cursor in the target chat input field, press Alt + T to open the translation window, then click "Lock" to lock the current chat window.

💡 Window name appears at the top when locked
3

Type, Translate, Send

Type your content in the input field and press Enter. The translation will be automatically pasted to the locked chat window and sent.

💡 Auto-send can be toggled in settings

FAQ

Quick answers to common questions

How do I get an API Key? ▼
We recommend DeepSeek: Visit platform.deepseek.com, create an account, and generate an API Key in the console. DeepSeek offers free credits and affordable pricing. You can also use OpenAI, Tongyi Qianwen, or any other OpenAI-compatible API.
Which chat apps are supported? ▼
All Windows desktop chat applications are supported, including but not limited to: DingTalk, WeChat, WeCom, WhatsApp, Telegram, Skype, AliWangwang, Qianniu, and more. Any text input field that accepts paste works.
How is the translation quality? ▼
Translation quality depends on your chosen AI model. We recommend GPT-4, DeepSeek, or similar large language models, which far exceed traditional machine translation in accuracy and can properly understand context and e-commerce terminology.
Is the software secure? ▼
The software runs entirely locally and doesn't collect any user data. Translation requests go directly to your configured AI provider (DeepSeek, OpenAI, etc.) without passing through any third-party servers. Your API Key is stored only on your local computer.
Can I customize the hotkey? ▼
Yes. Click the settings icon and navigate to the "Hotkeys" tab to customize the window activation shortcut. Combinations with Alt, Ctrl, and Shift are supported.
